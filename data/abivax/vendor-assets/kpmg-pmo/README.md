## Board-deck takeaways

Recommended use:
- Position KPMG as an `independent program support layer`, not the builder
- Stress that KPMG brings:
  - governance cadence
  - design/controls challenge
  - FR/US coordination
  - user adoption support

Avoid:
- Calling the proposal only a `PMO role`
- Showing `USD 350K flat fee` as if that is the source proposal unless commercial negotiations later narrowed the scope

Best concise board language:
- `KPMG submitted a March 2026 support proposal to provide governance, functional assurance, controls oversight, and change management around the NetSuite implementation.`
- `The source proposal totals EUR 370.1K excluding VAT and expenses across 302 man-days over a 9-month implementation window.`
- `Management’s recommended use of KPMG is as an execution-control layer around the system integrator, not as a second implementer.`
